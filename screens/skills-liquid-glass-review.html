🔒 What are Private Sources?
🏢 Connect a private GitHub repo (e.g. owner/repo containing SKILL.md files) to install your team's internal skills.
📁 Or point to a local directory on disk that contains SKILL.md files — ideal for monorepos.
🔑 GitHub private repos require a Personal Access Token with repo (read) scope. Tokens are stored in the macOS Keychain.
⬇ Once added, install skills from private sources via Skill Store → Import…

PromptHub CLI
The PromptHub CLI bridges the app with your AI agents (Claude Code, Cursor, Codex, and more). It reads and writes SKILL.md files in each agent's configuration directory, keeping your installed skills in sync without manual file management.
1
Install via Homebrew Recommended
Install the PromptHub CLI with one command. Requires Homebrew (brew.sh).
Terminal
brew install prompthub/tap/prompthub-cli
2
Manual install Alternative
Download the binary directly and move it into your PATH.
Terminal
curl -fsSL https://prompthub.app/install.sh | sh
3
Verify installation
Confirm the CLI is on your PATH and shows the expected version.
Terminal
prompthub --version # prompthub-cli 1.4.2 (darwin/arm64)
4
Grant file access for your agents
PromptHub needs your permission to read and write each agent's hidden configuration folder. Click Grant Access for the agents you use. Permissions are stored as macOS security-scoped bookmarks.
